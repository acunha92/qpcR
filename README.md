# qpcR
Repository for the qpcR package. Work in progress. 

Code written by Alexander Cunha


To install: 

	install.packages('devtools')

	library(devtools)

	devtools::install_github('acunha92/qpcR', subdir = 'qpcR')
	
	librar(qpcR)




Requires: tidyverse, reshape



	pcR.summary(),

	pcR.meanplot(),

	pcR.expression(),

	pcR.expressionplot()
